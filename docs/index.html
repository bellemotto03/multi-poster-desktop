What is Multi-Poster Desktop?

Multi-Poster Desktop is a cross-platform desktop application that allows content creators and marketers to upload and publish video content to multiple social media accounts simultaneously from their PC. Built with security and efficiency in mind, it uses official APIs to ensure safe and compliant content publishing.

Key Features

🎬

Video Upload

Upload MP4 videos with chunked transfer for reliable large file handling

👥

Multi-Account

Manage multiple accounts from a single interface with secure OAuth authentication

🔒

Secure

Encrypted credential storage and official API integration

📊

Real-time Progress

Track upload progress and publishing status for each account

How It Works

  1. Connect Your Account

    Sign in securely via OAuth through the official API. Your credentials are encrypted locally.

  2. Select a Video

    Choose an MP4 video file from your computer. The app handles chunked uploads automatically.

  3. Add Details

    Enter a description and configure privacy settings before publishing.

  4. Publish

    Select target accounts and start the upload. Monitor real-time progress in the dashboard.

Supported Platforms

Multi-Poster Desktop works on macOS, Windows, and Linux. Requires Python 3.11+.
